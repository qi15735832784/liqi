# 框架介绍

面试官您好，我叫李琦，我上家公司是广州早丰投资控股有限公司，我们公司主要业务是电商，我们公司的产品有搜款网以及搜款app，我们公司每天有 10 万的 IP 量，100 万 pv，峰值在 13 万左右
主要使用了 nginx，tomcat，mysql，MinIO等服 务，总共有 32台服务器
我们公司的 app 有 15 万的 UV，200 万的用户量

我们公司的架构流程是这样的：我们有着两台溯源服务器，使用了 minio 搭配 CDN ，CDN使用的是七牛的。

前端服务器总共使用了4台服务器，其中位于两台位于北京，两台位于杭州，均使用了 nginx+php+minio 的架构，
web后端我们使用了三台服务器，一台北京，一台杭州，一台上海，也使用了 nginx+php+minio 的架构

然后是登录集群，我们使用了两台 LDAP 服务器，使用了 nginx，tomcat 的架构

智能推荐集群，使用了四台服务器，分别是前后端服务器以及 redis 缓存服务器和 mysql 服务器

手机集群使用了三台服务器，使用了 nginx+tomcat 的架构

我们还使用了三台 rabbitMQ 服务器组成了消息队列集群

微服务架构使用了五台服务器，分别是首页，新品，找档口，进货车，我的，使用了 tomcat 的架构

索引集群我们使用了五台ElasticSearch 服务器

还有缓存集群与数据库集群，缓存集群使用了三台 redis 组成了一主两从的架构
